# DSA Problem Suggester

This is a web application that uses the Gemini API to suggest Data Structures and Algorithms problems, track user progress, and provide a quiz mode.

## How to Run This Project

This is a front-end-only project. To run it, you must provide your own Gemini API key.

1.  **Download the Code:**
    Download this project as a ZIP or clone the repository.

2.  **Add Your API Key:**
    Paste your Gemini API key into `scipt.js` in the following format:
    ```javascript
    const API_KEY = "YOUR_API_KEY_GOES_HERE";
    ```

3.  **Run the project:**
    Open the `index.html` file in your browser (using a Live Server is recommended).
